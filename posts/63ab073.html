Lamada 表达式到底什么呢？一般指匿名函数，常有以下两种形式：

  • () -> ~，比如 x -> System.out.println(x);
  • () -> { }，比如 (a, b) -> { int x = a + b; return x;}

Lamada 表达式有什么用呢？一般用于实现函数式接口，即用匿名函数来表示接口参数。主要有以下两种情况：

  • 接口中只有一个方法需要重写，比如 new Thread(() -> a++; ).start();
  • Java 8 自带函数式接口：Predicate、 Function<T,R>、 Supplier、 Consumer 和BinaryOperator（在java.util.function包）

好了，要理解的东西基本就这些，下面就演示一下 stream流+lamada 进行集合操作….
